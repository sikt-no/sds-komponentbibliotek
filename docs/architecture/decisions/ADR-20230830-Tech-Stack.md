# Tech Stack

## Context and Problem Statement

Sikt designsystem needs various technology for building, testing, maintaining and distribution.

## Decision Outcome

### Testing

- Playwright **Do**

### Releases

- Standard Version **Maybe**  
  Unmaintained.
- Sematic Release **Don't**  
  No CLI.
- Release It

### Packages

- esbuild **Maybe**  
  Didn't give faster build when all plugins where added.
- tsup **Maybe**  
  Only experimental support for CSS

### Docs

- Astro **Maybe**
- 11ty **Maybe**
- Gatsby **Maybe/Don't**
