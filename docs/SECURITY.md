## Contact

Report vulnerabilities on [Slack](https://sikt-no.slack.com/archives/C04K82KES0J).

## Dependencies

No execution of lifecycle scripts.  
Minimum release age is default 5 days.

### Updates

A dual weekly scheduled automatic update is run by [Renovate bot](https://gitlab.sikt.no/platon/renovate/instances/designsystem/-/pipeline_schedules).

### Scan

All pipelines run a scan for vulnerabilities with Trivy.  
A weekly scheduled scan for vulnerabilities is done in [Gitlab](https://gitlab.sikt.no/designsystem/sds-komponentbibliotek/-/pipeline_schedules) and reports on failure to [Slack](https://sikt-no.slack.com/archives/C05P43F943A).

### SBOM

A daily SBOM (CycloneDX) is generated by Platon and distributed to [Dependency Track](https://platon.sikt.no/tjenester/dependency_track).
