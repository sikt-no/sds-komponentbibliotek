# `@sikt/sds-table`

## Consume

```sh
npm i -s @sikt/sds-table
```

### React

```js
import { Table } from "@sikt/sds-table";
import "@sikt/sds-table/dist/index.css";

<Table />;
```

### Stylesheets & custom markup

Import stylesheet:

```css
@import url("@sikt/sds-table");
```

Create custom markup:

```html
<div class="sds-table"></div>
```
