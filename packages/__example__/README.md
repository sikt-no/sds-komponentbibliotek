# `@sikt/sds-example`

## Consume

```sh
npm i -s @sikt/sds-example
```

### React

```js
import { Example } from "@sikt/sds-example";
import "@sikt/sds-example/dist/index.css";

<Example />;
```

### Stylesheets & custom markup

Import stylesheet:

```css
@import url("@sikt/sds-example");
```

Create custom markup:

```html
<div class="sds-example"></div>
```
