# `@sikt/sds-tag`

## Consume

```sh
npm i -s @sikt/sds-tag
```

### React

```js
import { TagStatus, TagCategory } from "@sikt/sds-tag";
import "@sikt/sds-tag/dist/index.css";

<TagStatus>My Status</TagStatus>;
<TagCategory>My Category</TagCategory>;
```
