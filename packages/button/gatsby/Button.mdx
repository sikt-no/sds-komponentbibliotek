import { Button, ButtonGroup } from "@sikt/sds-button";
import { ArrowCircleRightIcon } from "@sikt/sds-icons";

Pakken inneholder komponenterne `<Button />` og `<ButtonLink />`. ButtonLink er til for bruk der knappen endrer stien i nettleseren og blir til en vanlig hyper lenke.

### Varianter

<ButtonGroup variant="left">
  <Button variant="strong">Strong</Button>
  <Button>Subtle</Button>
  <Button variant="transparent">Transparent</Button>
  <Button variant="critical">Critical</Button>
</ButtonGroup>

### Størrelser

<div>
  <Button>Default</Button>
</div>
<div>
  <Button size="small">Small</Button>
</div>

### Med ikoner

#### Default

<div>
  <Button icon={<ArrowCircleRightIcon />}>Default</Button>
</div>

#### Left

<div>
  <Button icon={<ArrowCircleRightIcon />} iconVariant="left">
    Left
  </Button>
</div>

#### Only

Husk att du må ha et tilgjengeligt navn også på denna knapp. Bruk `children` eller `aria-label` for å sett det.

<div>
  <Button icon={<ArrowCircleRightIcon />} iconVariant="only">
    Only
  </Button>
</div>

### Gruppert

Grupper knapper sammen med komponenten `<ButtonGroup />`.

</ButtonGroup>
